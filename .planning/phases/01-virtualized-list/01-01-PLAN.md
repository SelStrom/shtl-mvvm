         - `private float _fixedHeight` -- 0 если переменная высота

       Конструктор/метод инициализации:

       - `void Rebuild(int itemCount, float fixedHeight)` -- для фиксированной высоты, заполняет prefix sum
       - `void Rebuild(int itemCount, Func<int, float> heightProvider)` -- для переменной высоты

       - `[MethodImpl(AggressiveInlining)] float GetItemHeight(int index)` -- _fixedHeight > 0 ? _fixedHeight : _prefixHeights[index+1] - _prefixHeights[index]
       - `float TotalHeight` -- property: _fixedHeight > 0 ? _itemCount * _fixedHeight : _prefixHeights[_itemCount]
       - `VisibleRange FindVisibleRange(float scrollPosition, float viewportHeight, int overscanCount)` -- binary search для первого/последнего видимого, расширение на overscan, clamping.        internal readonly struct VisibleRange
       ```csharp
       {
           public readonly int FirstIndex;
           public readonly int LastIndex; // inclusive
           public readonly int Count;
       }
       ```

       Binary search (для переменной высоты):
       ```
       lo=0, hi=_itemCount; while (lo < hi) { mid = lo + (hi-lo)/2; if (_prefixHeights[mid+1] <= scrollPos) lo = mid+1; else hi = mid; }
       ```

       Обеспечить предаллокацию _prefixHeights с запасом (capacity). При Rebuild если capacity не хватает -- Array.Resize.
       Нет LINQ, нет boxing, нет аллокаций в FindVisibleRange.

       Стиль: Allman braces, 4 spaces, `[MethodImpl(MethodImplOptions.AggressiveInlining)]` на hot path.
