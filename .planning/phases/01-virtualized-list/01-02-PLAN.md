    }

    - `[SerializeField] private Scrollbar _scrollbar;` (nullable, опциональный)

    - `[SerializeField] private float _elasticity = 0.1f;`

    - `private float _scrollPosition;`
    - `private float _velocity;`
    - `private float _contentHeight;`
    - `private bool _isDragging;`
    - `private float _prevDragPosition;`
    - `private Action<float> _onScrollPositionChanged;` -- callback для VirtualCollectionBinding

    **Публичные свойства (per UI-SPEC):**
    - `public float ScrollPosition { get => _scrollPosition; set { _scrollPosition = value; ClampScrollPosition(); OnScrollPositionChanged(); } }`
    - `public float Velocity => _velocity;`
    - `public int OverscanCount { get => _overscanCount; set => _overscanCount = value; }`
    - `public float ViewportHeight => _viewport != null ? _viewport.rect.height : 0f;`

    **Публичные методы:**
    - `void SetContentHeight(float height)` -- вызывается биндингом при пересчёте layout
    - `void ScrollTo(float position)` -- программный скролл (с инерцией? нет -- мгновенный)
    - `void ScrollToIndex(int index, Func<int, float> getOffset)` -- программный скролл к элементу. getOffset предоставляется биндингом (LayoutCalculator.GetItemOffset)
    - `void SetOnScrollPositionChanged(Action<float> callback)` -- регистрация callback из биндинга
    - `void ResetScroll()` -- scroll position = 0, velocity = 0

    **Event handlers:**
    - `OnBeginDrag(PointerEventData)`: `_isDragging = true; _velocity = 0f; _prevDragPosition = GetLocalPosition(eventData);`
    - `OnDrag(PointerEventData)`: `var pos = GetLocalPosition(eventData); var delta = pos - _prevDragPosition; _prevDragPosition = pos;` Apply delta к _scrollPosition. За границами -- RubberDelta. `OnScrollPositionChanged();`
    - `OnEndDrag(PointerEventData)`: `_isDragging = false;` Velocity уже накоплена в OnDrag через `_velocity = delta / Time.unscaledDeltaTime;`
    - `OnScroll(PointerEventData)`: `var delta = eventData.scrollDelta.y * _scrollSensitivity; _scrollPosition -= delta; ClampScrollPosition(); OnScrollPositionChanged();`

    **LateUpdate:**
    - Если `!_isDragging && _velocity != 0f`:
      - Если offset != 0 (за границами) и _movementType == Elastic: `_scrollPosition = Mathf.SmoothDamp(_scrollPosition, _scrollPosition - offset, ref _velocity, _elasticity, Mathf.Infinity, Time.unscaledDeltaTime);`
      - Иначе если _inertia: `_velocity *= Mathf.Pow(_decelerationRate, Time.unscaledDeltaTime); _scrollPosition += _velocity * Time.unscaledDeltaTime;`
      - Если `Mathf.Abs(_velocity) < 1f`: `_velocity = 0f;`
      - `ClampScrollPosition();` (для Clamped mode)
      - `OnScrollPositionChanged();`

    **Вспомогательные методы:**
    - `[MethodImpl(AggressiveInlining)] private float CalculateOffset()`: возвращает 0 если в пределах, иначе смещение за границу (scrollPos < 0 или scrollPos > maxScroll)
    - `[MethodImpl(AggressiveInlining)] private static float RubberDelta(float overStretching, float viewSize)`: формула из UI-SPEC
    - `private void ClampScrollPosition()`: для Clamped mode -- clamp к [0, maxScroll]. MaxScroll = max(0, _contentHeight - ViewportHeight)
    - `private float GetLocalPosition(PointerEventData eventData)`: RectTransformUtility.ScreenPointToLocalPointInRectangle
    - `[MethodImpl(AggressiveInlining)] private void OnScrollPositionChanged()`: вызывает `_onScrollPositionChanged?.Invoke(_scrollPosition);` + обновляет scrollbar если не null

    **Scrollbar (per UI-SPEC):**
    - В `OnScrollPositionChanged()`: если `_scrollbar != null` и `_contentHeight > ViewportHeight`: `_scrollbar.size = ViewportHeight / _contentHeight; _scrollbar.value = _scrollPosition / maxScroll;`
    - В `Awake()` или `OnEnable()`: подписка на `_scrollbar.onValueChanged` для bidirectional sync

    Стиль: Allman braces, 4 spaces, private fields с `_`, var everywhere. Нет LINQ. Нет аллокаций в LateUpdate, OnDrag, OnScroll.
